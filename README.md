# Agentic Equity Research

Buy-side investment research memos generated autonomously by an agentic AI system.

The system ingests raw SEC filings (10-Ks, 10-Qs) and earnings call transcripts, then produces a full research memo end-to-end — analytical framework, tables, charts, scenario analysis, and recommendation. No human editing, no templates.
